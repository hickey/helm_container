Helm Charts Repository

To use this Chart repository, please setup a connection to the repository with the following command:

$ helm repo add <REPO_NAME> %SCHEME%://%HTTP_HOST%/

Uploading Charts to the Repository

Charts can be uploaded either with the form below or by executing the following command:

$ curl -X POST -F file=@<FILENAME> %SCHEME%://%HTTP_HOST%/upload

It is required that the form key be file and the @ sign needs to be used to tell curl to send the contents of the file.
